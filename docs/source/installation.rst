*************
Dependencies
*************

These are the core dependencies that you will need for cytomulate. They should
be automatically installed with ``pip`` or ``conda``, but if there is an issue,
you can elect to install on your own.

* scikit-learn
* numpy
* scipy
* networkx
* matplotlib
* tqdm
